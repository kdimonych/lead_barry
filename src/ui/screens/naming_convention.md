# Naming Conventions

This module follows specific naming conventions:

| Prefix | Description          | Example       |
| ------ | -------------------- | ------------- |
| `Sc`   | Screen type          | [`ScWelcome`] |
| `Scv`  | Screen variable type | [`ScvState`]  |

Screen types follow the pattern: `Sc` + ScreenName
