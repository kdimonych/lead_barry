# lead_barry

__The lead-acid battery management controller written in Rust for Raspberry Pi Pico.__

> Capability:
>
> - Battery Charge/Discharge control(BMS)
> - WiFi connectivity
> - LoRa connectivity
> - Remote or by script DC Load control
> - Configuration and control through Web Ui / RestAPI
